# Jac21.github.io

Personal static site for [Jeremy Cantu](https://jac21.github.io/).

## Overview

The homepage is a minimal landing page built with plain HTML and CSS. It features:

- A left-aligned list of links for Resume, Portfolio, GitHub, LinkedIn, Photography, and Music
- A large editorial image on the right sourced from `public/images/content.jpg`
- A responsive layout that stacks cleanly on smaller screens

## Structure

- `index.html` contains the main landing page markup
- `public/styles/style.css` contains the site styling for the homepage
- `public/images/` contains the image assets used by the site
- `viz.html` and related `public/scripts/` assets remain in the repo for the skills visualization page

## Local Preview

Because this is a static site, you can preview it with any simple local server from the repo root, for example:

```bash
python3 -m http.server
```
